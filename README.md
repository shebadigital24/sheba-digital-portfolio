# SHEBA DIGITAL SOLUTIONS | Professional Portfolio

![Sheba Digital Solutions](https://raw.githubusercontent.com/shebadigital/portfolio/main/public/og-image.png)

## Overview
A high-performance, futuristic portfolio website for **SHEBA DIGITAL SOLUTIONS**. Engineered with a modern hybrid design philosophy, this project showcases our expertise in digital engineering, AI automation, and scalable cloud solutions.

## 🚀 Technologies Used
- **Frontend**: React 19, Vite, Framer Motion
- **3D Graphics**: Three.js, React Three Fiber, React Three Drei
- **Styling**: Vanilla CSS (Modern CSS Variables, Glassmorphism)
- **Icons**: Lucide React
- **Deployment**: Netlify

## 📁 Project Structure
The project follows a scalable and organized architecture:
- `src/components/common`: Reusable UI components (Navbar, Footer, CTA).
- `src/components/sections`: Individual page sections for modularity.
- `src/components/canvas`: 3D elements and Three.js scenes.
- `src/constants`: Centralized project data and configuration.
- `src/layouts`: Page layouts for consistent structure.
- `src/assets`: Static assets like images and styles.

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/shebadigital/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Run the development server:
```bash
npm run dev
```

### Production Build
Build the project for production:
```bash
npm run build
```
The output will be in the `dist/` directory.

## 🌐 Deployment

### Deploying to Netlify
1. Connect your GitHub repository to Netlify.
2. Use the following build settings:
   - **Build Command**: `npm run build`
   - **Publish Directory**: `dist`
3. Netlify will automatically handle routing via the included `netlify.toml`.

### GitHub Deployment
1. Initialize git:
   ```bash
   git init
   ```
2. Add all files:
   ```bash
   git add .
   ```
3. Commit changes:
   ```bash
   git commit -m "chore: initial deployment preparation"
   ```
4. Push to your repository.
